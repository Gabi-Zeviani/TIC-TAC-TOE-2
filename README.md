Este jogo foi criado para o projeto final da disciplina de Introdução à Programação I.

Criado em linguagem C. Contém ponteiros, arrays, funções, structs, arquivo de cabeçalho, tudo necessário para o funcionamento do jogo.

Você pode jogar contra uma pessoa ou contra o computador.

O jogo possui função de salvar o jogo.
